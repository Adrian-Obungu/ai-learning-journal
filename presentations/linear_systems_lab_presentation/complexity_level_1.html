Complexity Level 1: Basic Solution (NumPy Solver)

Objective

Quickly find the solution to a system of linear equations using NumPy's highly optimized np.linalg.solve function. This abstracts the underlying complexity for production-ready efficiency.

Implementation

import numpy as np

A = np.array([[1, 2, 2], [1, 3, 3], [2, 6, 5]])
B = np.array([4, 5, 6])

# Optimized library call
solution = np.linalg.solve(A, B)
print(f"Solution: {solution}")
