Complexity Level 2: Conceptual Span Check (Augmented Matrix)

Objective & Method

Show that a target vector b is in the span of vectors v1...vk iff the vector equation has a solution, which corresponds to the augmented matrix [A | b] being consistent.

  • Objective: link vector equations and augmented matrices.
  • Method: form A from the vectors, augment with b, and test consistency.

Rank Condition

The system is consistent precisely when rank(A) == rank([A | b]).

A = np.array(vectors).T augmented = np.hstack((A, target_vector.reshape(-1, 1))) rank_A = np.linalg.matrix_rank(A) rank_aug = np.linalg.matrix_rank(augmented) if rank_A == rank_aug: # consistent: b is in the span else: # inconsistent: b is not in the span

Applications

Useful for debugging linear systems, feature engineering, and explaining model outputs.

  • Debugging why a system has no solution.
  • Understanding feature dependencies.
  • Interpreting how inputs combine to give outputs.
