Basic Vector Operations with NumPy

Leverage NumPy for efficient, high-performance vector arithmetic. This level focuses on direct application of operations.

Vector Addition:

import numpy as np v1 = np.array([1, 2]) v2 = np.array([3, 4]) sum_v = v1 + v2 # [4, 6]

Scalar Multiplication:

scalar = 2 scaled_v = scalar * v1 # [2, 4]

Linear Combination:

lc_result = 2*v1 - 1*v2 # [-1, 0]

Basic Span Check

Quickly determine if a target vector can be formed by a linear combination of a set of spanning vectors using NumPy's optimized linear solver.

def is_in_span(vectors, target_vector): A = np.array(vectors).T try: # Solve Ax = b for coefficients coefficients = np.linalg.solve(A, target_vector) return np.allclose(np.dot(A, coefficients), target_vector) except np.linalg.LinAlgError: return False # Example: span_v = [np.array([1,0]), np.array([0,1])] target = np.array([5,7]) print(is_in_span(span_v, target)) # True
