Core Logic: REF vs. RREF

Condition Row-Echelon Form (REF) Reduced Row-Echelon Form (RREF)
Rows of Zeros Must be grouped at the bottom. Must be grouped at the bottom.
Leading Entry Must be to the right of the leading entry above it. Must be to the right of the leading entry above it.
Column Below Pivot All entries below leading entry must be zero. All entries below leading entry must be zero.
Leading Value Any non-zero real number. Must be exactly 1.
Column Above Pivot Any non-zero real number. Must be exactly 0.

The stair-step pattern is a defining visual characteristic of echelon forms. Every matrix is row-equivalent to a unique matrix in RREF, making it a deterministic target for algorithms.

In REF, the leading entries define pivot positions, which are critical for identifying basic and free variables in a system of linear equations.
