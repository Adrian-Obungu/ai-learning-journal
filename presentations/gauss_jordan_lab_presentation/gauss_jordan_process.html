The Gauss-Jordan Elimination Process

Step 1: Forward Elimination (Gaussian)
Transform the augmented matrix into Row-Echelon Form (REF). Use row swaps and replacements to create zeros below each pivot, moving from the leftmost column to the right.
Step 2: Pivot Normalization
Scale each non-zero row so that the leading entry (pivot) becomes exactly 1. This simplifies the matrix and prepares it for the final reduction phase.
Step 3: Backward Elimination (Jordan)
Work from the bottom-most pivot upwards. Use row replacements to create zeros above each leading 1. This transforms the matrix into its unique Reduced Row-Echelon Form (RREF).
Step 4: Solution Extraction
Read the solution directly from the RREF. Identify basic variables (pivot columns) and free variables (non-pivot columns) to express the complete solution set.
