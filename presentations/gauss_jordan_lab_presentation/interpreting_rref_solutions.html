Interpreting RREF: Solutions & Variables

Unique Solution
[ 1 0 0 | a ]
[ 0 1 0 | b ]
[ 0 0 1 | c ]

A unique solution exists when every column of the coefficient matrix contains a pivot position. The system is consistent, and each variable is a basic variable with a fixed value.

Inconsistent
[ 1 0 2 | 0 ]
[ 0 1 3 | 0 ]
[ 0 0 0 | 1 ]

The system is inconsistent if a pivot exists in the augmented column. This results in a mathematical contradiction (e.g., 0 = 1), indicating no solution exists.

Infinite Solutions
[ 1 0 2 | 5 ]
[ 0 1 3 | 2 ]
[ 0 0 0 | 0 ]

Infinite solutions occur when the system is consistent but contains free variables (columns without pivots). Basic variables are expressed in terms of these parameters.

Basic: x1, x2
Free: x3
