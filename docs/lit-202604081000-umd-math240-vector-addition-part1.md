# Literature Note - UMD Math 240 - Vector Addition (1.3 Part 1)

## Summary
This session introduces the fundamental concept of a **vector** as an ordered list of real numbers, emphasizing its algebraic definition and notation as a column vector. It defines two primary operations: **vector addition** and **scalar multiplication**, both performed component-wise. The session highlights that these operations create new vectors of the same length, underscoring the creation of new mathematical objects and operations.

## Key Concepts
-   **Vector**: An ordered list of real numbers, typically written as a column vector. The order of entries is crucial.
    -   Notation: Square brackets for column vectors (e.g., `[0, 1]`).
    -   Length/Dimension: Referred to as a "two-vector," "three-vector," or "n-vector" based on the number of entries.
-   **Column Vectors**: The standard notation used, which integrates well with matrix operations.
-   **Vector Notation**: Vectors are denoted by a letter with a vector sign over top (e.g., $\vec{u}$, $\vec{v}$). Scalars (real numbers) are represented by plain letters (e.g., $c$, $d$).
-   **Vector Space $\mathbb{R}^n$**: The set of all possible column vectors of length $n$. For example, $\mathbb{R}^2$ is the set of all two-vectors, geometrically representing points or arrows in a 2D plane.
-   **Vector Addition**: Defined component-wise. If $\vec{v} = [a_1, \dots, a_n]$ and $\vec{w} = [b_1, \dots, b_n]$, then $\vec{v} + \vec{w} = [a_1+b_1, \dots, a_n+b_n]$. The result is another $n$-vector.
-   **Scalar Multiplication**: Defined component-wise. If $\vec{v} = [a_1, \dots, a_n]$ and $c$ is a scalar, then $c\vec{v} = [ca_1, \dots, ca_n]$. The result is another $n$-vector.
-   **New Operations**: The session stresses that vector addition and scalar multiplication are *new* operations, distinct from real number addition/multiplication, even though they use familiar symbols. Their properties must be proven from their definitions.
