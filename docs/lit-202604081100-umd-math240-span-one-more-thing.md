# Literature Note - UMD Math 240 - Span: One More Thing (1.3 Part 7)

## Summary
This concluding session for Module 1.3 provides a concise summary of the geometric implications of the **span of two vectors**, particularly when considering their linear dependence or independence. It clarifies that the span of two non-zero vectors can either be a line (if they are scalar multiples of each other) or a plane (if they are not scalar multiples of each other), always passing through the origin. This reinforces the visual understanding of how vectors combine to "fill" a subspace.

## Key Concepts
-   **Span of Two Non-Zero Vectors ($\vec{u}, \vec{v}$)**:
    -   **Case 1: Linearly Dependent (Collinear)**: If one vector is a scalar multiple of the other (e.g., $\vec{u} = c\vec{v}$), then their span is a **line** passing through the origin. All linear combinations $a\vec{u} + b\vec{v}$ will simply be another scalar multiple of $\vec{v}$ (or $\vec{u}$), effectively staying on the same line.
    -   **Case 2: Linearly Independent (Non-Collinear)**: If neither vector is a scalar multiple of the other, their span is a **plane** passing through the origin. This plane is formed by all possible linear combinations of the two vectors.
-   **Origin Inclusion**: The span of any set of vectors always includes the origin (the zero vector), as the trivial linear combination (all weights zero) always results in the zero vector.
-   **Subspace**: The span of a set of vectors always forms a subspace of the larger vector space $\mathbb{R}^n$. This means it is closed under vector addition and scalar multiplication.
