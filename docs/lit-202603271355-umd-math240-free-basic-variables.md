# Literature Note - UMD Math 240 - Free Variables and Basic Variables (1.2 Part 4)

## Summary
This session introduces the critical concepts of **basic variables** and **free variables**, which are essential for describing the solution set of a system of linear equations, especially when there are infinitely many solutions. It explains how these variables are identified from the Reduced Row-Echelon Form (RREF) of an augmented matrix and how to express the general solution in terms of the free variables, including the introduction of **parametric vector form**.

## Key Concepts
-   **Basic Variables**: Variables corresponding to **pivot columns** in the RREF of an augmented matrix. These variables are constrained and can be expressed in terms of free variables.
-   **Free Variables**: Variables corresponding to **non-pivot columns** (excluding the augmented column) in the RREF. These variables can take any real value, and their choice determines the specific solution within an infinite solution set.
-   **Interpretation of RREF**: The RREF directly reveals which variables are basic and which are free, allowing for the systematic construction of the solution set.
-   **Infinite Solutions**: The presence of one or more free variables indicates that the system has infinitely many solutions.
-   **Parametric Vector Form**: A powerful way to represent the entire solution set of a consistent system with free variables. It expresses the solution vector as a sum of a particular solution (when all free variables are zero) and a linear combination of vectors, each multiplied by a free variable (parameter). Geometrically, this describes a line, plane, or hyperplane.
